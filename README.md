# 📌 Project Overview

The **Zenetive Todo Application** is a modern **full-stack task management system** developed using the **MERN Stack (MongoDB, Express.js, React, Node.js)**.

This application enables users to **create, manage, update, and delete tasks efficiently** through a clean and responsive interface while maintaining persistent data storage through a scalable backend architecture.

The project demonstrates **RESTful API integration, dynamic frontend rendering, and database-driven task management**, making it a practical example of a modern full-stack web application.

---

# ✨ Key Features

✔ Create new tasks instantly
✔ Update task completion status
✔ Delete tasks dynamically
✔ Persistent task storage using MongoDB
✔ REST API powered backend
✔ Interactive and responsive UI
✔ Fast client-server communication
✔ Modular and scalable code architecture

# ⚙️ System Architecture

```
Client (React Frontend)
        │
        ▼
REST API (Express + Node.js)
        │
        ▼
MongoDB Database
```

The frontend communicates with the backend using **HTTP requests**, while the backend processes business logic and stores task data in **MongoDB**.

# 🚀 Installation & Setup

## 1️⃣ Clone the Repository

```
git clone https://github.com/Rishidevlx/zenetive-Task-1-TODO_APP.git
```

---

## 2️⃣ Navigate to Project Folder

```
cd zenetive-Task-1-TODO_APP
```

---

## 3️⃣ Install Dependencies

### Backend

```
cd server
npm install
```

### Frontend

```
cd client
npm install
```

---

## 4️⃣ Configure Environment Variables

Create a **.env file** inside the server folder

```
MONGO_URI=your_mongodb_connection_string
PORT=5000
```

---

## 5️⃣ Run the Application

### Start Backend

```
npm start
```

### Start Frontend

```
npm start
```

The application will run locally on:

```
http://localhost:3000
```
